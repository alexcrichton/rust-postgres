Struct postgres::PostgresConnectParams[src]

pub struct PostgresConnectParams {
    pub target: PostgresConnectTarget,
    pub port: Option<Port>,
    pub user: Option<StrBuf>,
    pub password: Option<StrBuf>,
    pub database: Option<StrBuf>,
    pub options: Vec<(StrBuf, StrBuf)>,
}

Information necessary to open a new connection to a Postgres server.

Fields

target

The target server

port

The target port.

Defaults to 5432 if not specified.

user

The user to login as.

PostgresConnection::connect requires a user but cancel_query does not.

password

An optional password used for authentication

database

The database to connect to. Defaults the value of user.

options

Runtime parameters to be passed to the Postgres backend.
