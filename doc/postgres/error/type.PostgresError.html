pub enum PostgresError {
    PgDbError(PostgresDbError),
    PgStreamError(IoError),
    PgStreamDesynchronized,
    PgWrongConnection,
    PgWrongParamCount {
        pub expected: uint,
        pub actual: uint,
    },
    PgWrongType(PostgresType),
    PgInvalidColumn,
    PgWasNull,
    PgWrongTransaction,
}

An error encountered when communicating with the Postgres server

Variants

PgDbError

An error reported by the Postgres server

PgStreamError

An error communicating with the Postgres server

PgStreamDesynchronized

The communication channel with the Postgres server has desynchronized due to an earlier communications error.

PgWrongConnection

A prepared statement was executed on a connection it does not belong to

PgWrongParamCount

An incorrect number of parameters were bound to a statement

Fields

expected

The expected number of parameters

actual

The actual number of parameters

PgWrongType

An attempt was made to convert between incompatible Rust and Postgres types

PgInvalidColumn

An attempt was made to read from a column that does not exist

PgWasNull

A value was NULL but converted to a non-nullable Rust type

PgWrongTransaction

An attempt was made to prepare a statement or start a transaction on an object other than the active transaction
